# Lottery
## Introduction
It is a simple project that randomly generates six zodiac in twelve zodiacs every 5 minutes.

It works from 10.00 a.m. to 22.55 p.m.

We can use the background to manipulate the result in the latest period.

## Development tools
* Springboot
* Thymeleaf
* bootstrap
* Intellij IDEA
